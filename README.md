# MarketHub

MarketHub is a planned multi-vendor e-commerce marketplace for Africa, designed as a regional alternative to large marketplace platforms.

## Run the Web App

```bash
npm install
npm run dev
```

Local URL:

```text
http://127.0.0.1:5173
```

Production build:

```bash
npm run build
```
