# Vendor Dashboard Page

## Purpose

The vendor dashboard gives sellers a command center for managing sales, products, orders, payouts, and store settings.

## Primary Users

- Vendors.

## Key Sections

- Sales overview.
- Recent orders.
- Product management.
- Inventory alerts.
- Payout summary.
- Marketing tools.
- Store settings.

## Core Requirements

- Show total sales, revenue, orders, and conversion rate.
- List recent orders with fulfillment status.
- Highlight low stock products.
- Show earnings, pending payouts, commissions, and withdrawals.
- Allow navigation to products, orders, analytics, payouts, reviews, marketing, and store settings.

## MVP Data Needed

- Vendor profile.
- Orders.
- Products.
- Sales totals.
- Inventory levels.
- Payout records.
- Commission records.
- Review summary.
