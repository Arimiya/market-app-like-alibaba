# Stores Page

## Purpose

The stores page lets customers discover vendors, compare store ratings, and visit individual storefronts.

## Primary Users

- Customers.
- Vendors.
- Admins.

## Key Sections

- Store search.
- Store filters.
- Top stores grid.
- Store cards.
- Sorting by popularity, rating, newness, or product count.

## Core Requirements

- List approved vendor stores only.
- Show logo, store name, rating, product count, location, and verification status.
- Link each store to its vendor storefront.
- Support sorting and filtering.
- Hide suspended or unverified stores from public discovery.

## MVP Data Needed

- Vendor store name.
- Logo.
- Banner.
- Description.
- Rating.
- Product count.
- Location.
- Verification status.
