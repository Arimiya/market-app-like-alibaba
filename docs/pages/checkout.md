# Checkout Page

## Purpose

The checkout page completes the purchase with customer delivery information, payment selection, and final order confirmation.

## Primary Users

- Customers.

## Key Sections

- Delivery information form.
- Address selection.
- Delivery method.
- Payment method.
- Order summary.
- Place order action.

## Core Requirements

- Support Mobile Money, card, bank transfer, and wallet balance payments.
- Validate delivery address and contact details.
- Show final total before payment.
- Create orders per vendor where needed.
- Trigger payment verification.
- Send order confirmation notifications.

## MVP Data Needed

- Customer profile.
- Delivery address.
- Cart items.
- Payment method.
- Payment gateway transaction.
- Shipping fee.
- Tax.
- Order total.
