# Analytics

## Purpose

Analytics help vendors improve sales, customers receive better recommendations, and admins monitor marketplace health.

## Vendor Metrics

- Total sales.
- Revenue.
- Orders.
- Conversion rate.
- Best sellers.
- Product views.
- Inventory turnover.
- New customers.
- Repeat customers.
- Customer lifetime value.
- Earnings.
- Withdrawals.
- Pending payouts.
- Commissions.
- Campaign performance.
- Coupon usage.
- Traffic sources.

## Customer Analytics

- User behavior.
- Cart abandonment.
- Purchase history.
- Search trends.

## Admin Metrics

- Platform revenue.
- Total orders.
- Total vendors.
- Total customers.
- Refund rate.
- Dispute rate.
- Fraud flags.
- Commission revenue.

## MVP Requirements

- Track page views and product views.
- Track cart and checkout events.
- Track order and payment events.
- Show simple dashboards before advanced prediction features.
