# Payments and Payouts

## Purpose

Payments and payouts make customer checkout secure while ensuring vendors receive earnings correctly.

## Supported Customer Payments

- Visa.
- Mastercard.
- Mobile Money.
- Bank transfer.
- Apple Pay.
- Google Pay.
- Wallet balance.

## African Mobile Money

- MTN Mobile Money.
- Airtel Money.
- Telecel Cash.
- M-Pesa.

## Gateway Integrations

- Flutterwave.
- Paystack.
- Stripe.

## Security Requirements

- PCI-aware payment handling.
- 3D Secure where available.
- Fraud detection.
- Escrow-style payment holding.
- Payment verification before order fulfillment.

## Vendor Payout Rules

- Hold period before settlement.
- Refund reserve.
- Minimum withdrawal amount.
- Scheduled withdrawals.
- Mobile Money payouts.
- Bank payouts.
