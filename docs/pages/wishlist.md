# Wishlist Page

## Purpose

The wishlist page lets customers save products for future purchase.

## Primary Users

- Customers.

## Key Sections

- Saved product grid.
- Add to cart action.
- Remove from wishlist action.
- Share wishlist action.

## Core Requirements

- Allow authenticated customers to save products.
- Show product availability and current price.
- Allow quick add to cart.
- Notify customers when wishlist products go on sale if notification preferences allow it.

## MVP Data Needed

- Customer.
- Product.
- Current price.
- Stock status.
- Vendor.
- Date added.
