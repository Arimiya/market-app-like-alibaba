# Deals Page

## Purpose

The deals page highlights discounted products, flash sales, campaign offers, and time-limited promotions.

## Primary Users

- Customers.
- Vendors.
- Admins.

## Key Sections

- Deals tabs: all deals, flash deals, top deals, upcoming.
- Countdown timer for time-sensitive campaigns.
- Deal product grid.
- Discount badges.
- Campaign banners.

## Core Requirements

- Show only active deals to customers.
- Support start and end times.
- Allow vendors to create product-level promotions.
- Allow admins to create platform-wide campaigns.
- Display original price, discount price, discount percentage, and stock availability.

## MVP Data Needed

- Product details.
- Original price.
- Discount price.
- Discount percentage.
- Campaign name.
- Start and end date.
- Vendor.
- Stock quantity.
