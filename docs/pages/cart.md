# Cart Page

## Purpose

The cart page lets customers review selected products before checkout, including items from multiple vendors.

## Primary Users

- Customers.

## Key Sections

- Cart item list grouped by vendor.
- Quantity controls.
- Remove item action.
- Save for later action.
- Coupon input.
- Shipping estimate.
- Order summary.

## Core Requirements

- Support multi-vendor carts.
- Update quantity without losing cart state.
- Recalculate subtotal, discounts, shipping estimate, tax, and total.
- Prevent checkout for out-of-stock items.
- Allow customers to move items to wishlist or save for later.

## MVP Data Needed

- Cart items.
- Product price.
- Product discount.
- Product stock.
- Vendor.
- Shipping estimate.
- Coupon.
- Tax rule.
