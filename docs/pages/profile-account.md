# Profile and Account Page

## Purpose

The profile and account page lets users manage personal information, addresses, payment methods, order history, wallet, and security settings.

## Primary Users

- Customers.
- Vendors.
- Admins.

## Key Sections

- Account overview.
- Personal information.
- Address book.
- Payment methods.
- Order history.
- Wallet.
- Settings.
- Logout.

## Core Requirements

- Allow users to update profile information.
- Manage multiple delivery addresses.
- Manage saved payment methods where supported.
- Show order history and invoices.
- Support password changes and multi-factor authentication.
- Respect role-specific account sections.

## MVP Data Needed

- User profile.
- Addresses.
- Payment methods.
- Orders.
- Wallet balance.
- Security settings.
- Notification preferences.
