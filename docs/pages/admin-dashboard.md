# Admin Dashboard Page

## Purpose

The admin dashboard gives marketplace operators visibility and control over users, vendors, products, orders, finances, and risk.

## Primary Users

- Admins.
- Support staff.
- Finance staff.
- Risk and compliance staff.

## Key Sections

- Platform overview.
- User management.
- Vendor approvals.
- Product moderation.
- Order monitoring.
- Disputes and refunds.
- Financial reports.
- Fraud monitoring.
- Settings.

## Core Requirements

- Show platform revenue, total orders, total vendors, and total customers.
- Approve, reject, suspend, or reactivate vendors.
- Manage categories and products.
- Monitor order statuses and disputes.
- Review refund approvals.
- Manage commissions and payout rules.
- Identify suspicious accounts, transactions, and reviews.

## MVP Data Needed

- Users.
- Vendors.
- Products.
- Categories.
- Orders.
- Revenue.
- Commissions.
- Payouts.
- Refunds.
- Fraud flags.
