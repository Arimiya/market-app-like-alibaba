# Home Page

## Purpose

The home page introduces customers to MarketHub's best products, trusted vendors, promotions, and main shopping categories.

## Primary Users

- Customers.
- Guest visitors.

## Key Sections

- Top navigation with logo, search, account, cart, wishlist, and notification entry points.
- Hero promotion banner for seasonal offers.
- Category shortcut row.
- Flash deals section.
- Best sellers section.
- New arrivals section.
- Featured stores section.

## Core Requirements

- Show marketplace-wide promotions.
- Support keyword search from the header.
- Link users to categories, stores, product pages, deals, cart, wishlist, and account.
- Display product cards with image, name, price, discount, rating, vendor, and add-to-cart action.
- Support responsive desktop and mobile layouts.

## MVP Data Needed

- Featured products.
- Categories.
- Active deals.
- Featured vendors.
- Customer cart count.
- Customer wishlist count.
