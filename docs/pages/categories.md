# Categories Page

## Purpose

The categories page helps customers browse MarketHub by product type and quickly move into filtered product listings.

## Primary Users

- Customers.
- Guest visitors.

## Key Sections

- Search and navigation header.
- Category grid.
- Category icons or images.
- Product count per category.
- Optional featured category banners.

## Core Requirements

- Show all active product categories.
- Support parent and child categories.
- Link each category to a filtered product listing.
- Allow admins to control category visibility and ordering.
- Display category product counts.

## MVP Data Needed

- Category name.
- Category slug.
- Category icon or image.
- Parent category.
- Product count.
- Active status.
