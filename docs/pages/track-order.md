# Track Order Page

## Purpose

The track order page allows customers to check delivery progress and view recent orders.

## Primary Users

- Customers.
- Support admins.

## Key Sections

- Order number search.
- Recent orders list.
- Order status badges.
- Shipment tracking details.
- Link to order detail and invoice.

## Core Requirements

- Allow customers to search by order number.
- Show order status: pending, paid, processing, packed, shipped, delivered, returned, refunded, or cancelled.
- Show delivery estimate and tracking events when available.
- Restrict customers to their own orders.
- Allow support/admin users to search broader order records.

## MVP Data Needed

- Order number.
- Customer.
- Vendor.
- Status.
- Total amount.
- Shipping carrier.
- Tracking number.
- Delivery estimate.
- Timeline events.
